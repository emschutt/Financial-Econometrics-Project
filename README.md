# Financial-Econometrics-Project
This repository presents an empirical financial econometrics study focused on modeling and forecasting financial time series using advanced econometric techniques. The project applies quantitative methods to analyze market dynamics, estimate predictive models, and evaluate their statistical and economic significance.

## Project Overview

The objective of this project is to investigate financial time-series behavior through econometric modeling, with emphasis on:

- Time-series stationarity diagnostics
- Model estimation for financial returns and volatility
- Forecasting asset price dynamics
- Model comparison using econometric performance metrics

The analysis combines theoretical financial econometrics concepts with practical implementation in Python.

## Methodology

The project follows a standard econometric workflow:

1. **Data Collection & Preprocessing**
   - Import and clean financial market data
   - Handle missing values and outliers
   - Compute log-returns where appropriate

2. **Exploratory Data Analysis**
   - Descriptive statistics
   - Distributional analysis
   - Visualization of time-series behavior

3. **Stationarity Testing**
   - Augmented Dickey-Fuller (ADF) tests
   - Unit root diagnostics

4. **Econometric Modeling**
   Depending on the specification, models may include:
   - AR / ARIMA models
   - GARCH-family volatility models
   - Regression-based forecasting models

5. **Model Evaluation**
   - AIC / BIC comparison
   - Residual diagnostics
   - Forecast accuracy metrics (RMSE, MAE)


## Key Outputs

* Econometric estimation of financial time-series models
* Volatility clustering analysis
* Forecast performance comparison across models
* Visual diagnostics and statistical inference
